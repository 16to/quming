## 快速上手 
1. 下载HBuilderX ![https://www.dcloud.io/hbuilderx.html]
2. 文件 -> 新建 -> 项目
3. 打开一个苹果模拟器 open -a Simulator
4. 下载微信小程序开发者工具 ![https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html]
5. 打开一个安卓模拟器
